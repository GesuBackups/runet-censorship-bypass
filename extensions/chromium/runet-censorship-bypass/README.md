# Install

Tested on:

NodeJS: v16.1.0.
NPM: 7.11.2.
OS: Linux Mint 20 Xfce Edition.

```
npm install
cd src/extension-common/pages/options/
npm install
cd -

# For debugging:
npm start
# Use your build/extension-beta

# For production:
npm start
# Use your build/extension-full or build/extension-mini
```
